# Home Assistant 榜威集成

榜威集成是一个由榜威官方提供支持的 Home Assistant 的集成组件，它可以让您在 Home Assistant 中使用榜威 IoT 智能设备。

## 安装

> Home Assistant 版本要求：
>
> - Core $\geq$ 2024.4.4
> - Operating System $\geq$ 13.0

### 使用 git clone 命令从 GitHub 下载

```bash
cd config
git clone https://github.com/bweetech/ha_bwee_home.git
cd ha_bwee_home
./install.sh /config
```
